Lo que vamos a construir

Algo parecido a esto

  * Comandos OS X / Linux

Instalación de la última versión de Meteor
$ curl https://install.meteor.com/ | sh
Creación de la app
$ meteor create simpleEvernote
$ cd simpleEvernote
Arrancamos Meteor
$ meteor

Meteor server en http://localhost:3000

Empezamos...

Otras ventajas

Hot Code Pushes

Actualiza tu app mientras los usuarios estan conectados sin interferir en ellos.

Nativo

Pasa a plataformas móvil usando unos simples comandos

$ meteor install-sdk 'ios/android'
$ meteor add-platform 'ios/andrioid'
$ meteor run 'ios/android' // arranca emulador 
$ meteor run 'ios-device/android-device' // dispositivo físico
